  - Section name: `SECTION_NAMES[sectionKey]` (from generation.ts)

Status mapping logic:
- If `editorState.content` is empty/null and status is not 'generating': status = 'missing'
- If `editorState.compliance_flags.length > 0`: status = 'needs-review'
- If `editorState.status === 'complete'` and no compliance flags: status = 'complete'
- If `editorState.status === 'generating'`: status = 'generating'
- If `editorState.status === 'error'`: status = 'error'
