Implement `workspaceReducer(state: WorkspaceState, action: WorkspaceAction): WorkspaceState` handling all action types defined in `workspace.ts`:

- `SET_SECTIONS` — replace `state.sections` with payload
- `SET_ACTIVE_SECTION` — set `state.active_section` to payload
- `UPDATE_CONTENT` — update `sections[section_key].content` with payload content
- `SET_LOCKED` — update `sections[section_key].is_locked`; also update Supabase `proposal_sections` row `is_locked` column (fire-and-forget, no await needed in reducer — dispatch a side effect or handle in component)
- `SET_AUTOSAVE_STATUS` — update `sections[section_key].autosave_status`
- `START_AI_ACTION` — set `sections[section_key].ai_action` with type, snapshot, streaming=true, preview_content=''
- `UPDATE_AI_PREVIEW` — append/set `sections[section_key].ai_action.preview_content`
- `COMPLETE_AI_STREAM` — set `sections[section_key].ai_action.streaming = false`
- `ACCEPT_AI_ACTION` — set `sections[section_key].content = sections[section_key].ai_action.preview_content`, clear `ai_action` to null
- `REJECT_AI_ACTION` — clear `sections[section_key].ai_action` to null
- `SET_COMPLIANCE_FLAGS` — set `sections[section_key].compliance_flags`
- `SET_COMPLIANCE_CHECKING` — set `sections[section_key].compliance_checking`
- `SET_CONSISTENCY_FLAGS` — set `state.consistency_flags`
- `DISMISS_CONSISTENCY` — set `state.consistency_dismissed = true`
- `OPEN_VERSION_HISTORY` — set `state.version_history_open` to payload (section_key)
- `CLOSE_VERSION_HISTORY` — set `state.version_history_open` to null

**File 2: `src/hooks/useAutosave.ts`**

```typescript
import { useRef, useCallback } from 'react'
import { supabase } from '../lib/supabase'

export function useAutosave(proposalId: string, sectionKey: string, onStatusChange: (status: 'idle' | 'saving' | 'saved') => void) {
  const timerRef = useRef<ReturnType<typeof setTimeout> | null>(null)

  const triggerAutosave = useCallback((html: string) => {
    if (timerRef.current) clearTimeout(timerRef.current)
    timerRef.current = setTimeout(async () => {
      onStatusChange('saving')
      const { error } = await supabase
        .from('proposal_sections')
        .update({ last_saved_content: html, updated_at: new Date().toISOString() })
        .eq('proposal_id', proposalId)
        .eq('section_key', sectionKey)
      onStatusChange(error ? 'idle' : 'saved')
    }, 1500) // 1500ms debounce per UI-SPEC
  }, [proposalId, sectionKey, onStatusChange])

  const cancel = useCallback(() => {
    if (timerRef.current) clearTimeout(timerRef.current)
  }, [])

  return { triggerAutosave, cancel }
}
```
